# not-tomorrow-today


Günlük yapılacakları hızlıca eklemek, görmek ve tamamlamak için hazırlanmış basit bir görev yönetim uygulaması.

Özellikler

Büyük dijital saat (saat–dakika–saniye)

Yapılacaklar listesi ekleme / silme / tamamlama

Tamamlanan görevler için ayrı alan

Renkli kartlar ve emoji destekli motivasyon

Veriler LocalStorage’da saklanır

Mobil ve masaüstü uyumlu sade tasarım
